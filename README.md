# LiveChat
## 大三下学期期末课程设计，web即时聊天应用
+ 支持发送静态以及gif表情。
+ 支持窗口抖动。
+ 默认为群聊模式，支持将消息发送给特定用户。
+ 在线用户面板实时更新。
+ 支持发送自定义图片。（）
+ 各种动画效果（velocity.js实现）。
+ 具有良好的可扩展性。
